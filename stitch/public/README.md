# RAEN site images (upload here)

Put all media files you want to use on the website inside:

- `stitch/public/images/`

Then reference them from your HTML like:

- `public/images/your-file.jpg`

Notes:

- Keep filenames lowercase and hyphenated (example: `atelier-hero.jpg`).
- Prefer WebP/JPG for photos, SVG for logos/graphics.
